# Mariana Auth Demo
A simple example of how to use Mariana Tek Universal Authentication.

## System Dependencies
- Node.js v14.2
- nvm
- [foreman](https://github.com/ddollar/foreman) or similar (e.g. forego, node-foreman, honcho)

## Installation
```bash
$ nvm install
$ npm install
```

## Local Development
```bash
$ nvm use
$ npm run dev
```
